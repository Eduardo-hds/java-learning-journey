# 📌 Stage 1 — Student Data Input (START HERE)

## Objective

Create a program that collects basic student information and prints it back.

---

## Requirements

Create a file:

```text
StudentSystem.java
```

Ask the user for:

* Full name (`String`)
* Age (`int`)
* Grade 1 (`double`)
* Grade 2 (`double`)

Then display everything in a formatted output.

---

## Example Output

```text
Enter your name: Eduardo
Enter your age: 20
Enter grade 1: 8.5
Enter grade 2: 7.0

--- Student Data ---
Name: Eduardo
Age: 20
Grade 1: 8.5
Grade 2: 7.0
```

---

## Rules

* Use `Scanner`
* Use proper naming conventions
* Use appropriate data types
* Close the scanner
* Handle `nextLine()` correctly after numbers

---

## Constraints (IMPORTANT)

Do NOT calculate anything yet.

Do NOT use constants yet.

Do NOT use loops or conditions.

Keep it simple input/output only.
