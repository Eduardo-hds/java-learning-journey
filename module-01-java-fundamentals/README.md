# Module 01 — Java Fundamentals

## Overview

Welcome to **Module 01** of the Java Bootcamp.

This module introduces the Java programming language itself. Here, you'll learn the fundamental concepts that every Java developer must master before moving on to Object-Oriented Programming and more advanced topics.

By the end of this module, you'll be able to write complete console applications, understand how Java executes programs, work with variables and data types, perform calculations, receive user input, debug simple applications, and write clean, professional code.

---

## Learning Objectives

During this module, you will learn how to:

- Understand the structure of a Java program
- Write and execute Java applications
- Work with variables and memory
- Use Java primitive data types
- Convert values between different types
- Declare constants using `final`
- Use arithmetic, relational, logical, and assignment operators
- Build expressions correctly
- Receive user input from the console
- Write useful comments and documentation
- Follow Java naming conventions
- Debug simple programs
- Develop clean and readable console applications

## Skills You Will Acquire

After completing this module, you will be able to:

- Create Java programs from scratch
- Understand how Java executes code
- Store and manipulate data efficiently
- Read data entered by users
- Perform calculations using Java operators
- Write cleaner and more maintainable code
- Identify and fix common programming errors
- Apply Java coding conventions used by professional developers

---

## Final Project

Throughout the module, you will progressively build a console application that combines every concept studied.

The project is developed chapter by chapter, allowing each new concept to be applied immediately in practice.

---

## Prerequisites

Before starting this module, you should already know:

- How Java works at a high level
- How to install Java (JDK)
- How to use IntelliJ IDEA
- How to compile and run Java applications

These topics were covered in **Module 00 — Development Environment**.

---

## Goal

The objective of this module is not only to learn Java syntax, but also to understand **why the language works the way it does**.

Every topic will be explored from first principles, emphasizing internal behavior, best practices, common mistakes, and professional coding standards.

This foundation will prepare you for the next module, where you'll begin learning Object-Oriented Programming.
