# ☕ Java Bootcamp

A complete and structured Java Bootcamp documenting my journey from the fundamentals of the language to professional backend development with Spring Boot.

This repository is organized as a long-term learning project, combining theory, practical exercises, challenges, and real-world projects. Each module is designed to build upon the previous one, following a progressive learning path inspired by the DevDojo Java course and enriched with additional explanations, best practices, and hands-on experience.

The goal is not only to learn Java syntax, but to develop a deep understanding of the language, its ecosystem, and the concepts used in professional software development.

---

# 🎯 Objectives

- Build a solid foundation in Java.
- Master Object-Oriented Programming.
- Learn modern Java features and APIs.
- Develop problem-solving skills through exercises and challenges.
- Create practical projects for each module.
- Learn backend development with JDBC and Spring Boot.
- Apply clean code principles and best practices throughout the learning process.

---

# 📂 Repository Structure

Each module is organized in its own directory and contains:

- A dedicated Java project.

- Programming challenges.

│

# 💡 Learning Philosophy

This repository is focused on learning by doing.

Every module follows the same methodology:

- Study the concepts.
- Understand how they work internally.
- Practice with guided exercises.
- Solve independent challenges.
- Build a complete project.
- Review the content before moving forward.

The objective is to understand not only _how_ something works, but also _why_ it exists and _when_ it should be used.

---

## 🤖 AI-Assisted Learning

This Bootcamp is managed with the support of ChatGPT as a personal mentor and learning coordinator.

Instead of using AI to generate ready-made solutions, it is used to:

- Design and organize the learning roadmap.
- Explain concepts in depth.
- Create practical exercises and coding challenges.
- Review solutions and provide detailed feedback.
- Propose real-world projects.
- Reinforce best practices and clean code principles.
- Track progress across all modules.

The goal is to use AI as a teaching assistant, promoting active learning, critical thinking, and long-term knowledge retention rather than simply generating code.
