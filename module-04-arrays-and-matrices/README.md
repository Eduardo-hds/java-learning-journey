# 🚀 Final Project Proposal

## 🎓 Student Grade Management System (Console Application)

You will build a progressive system that manages student grades using arrays and matrices.

---

## Core Features:

### 📌 Data Storage

* Store multiple students’ grades using arrays
* Organize grade sets efficiently

### 📌 Calculations

* Average per student
* Overall class average
* Highest and lowest grades

### 📌 Search System

* Find a specific grade or student index

### 📌 Reports

* Display formatted grade reports
* Show statistics summary

### 📌 Matrix Component

* Represent class data in a matrix format:

  * rows = students
  * columns = exams/assignments

* Perform:

  * row sum (student total)
  * column sum (exam performance)

---

## Progression Rule:

The project will NOT be built at once.

Each chapter will incrementally add functionality until the final system is complete.

---

# ⚠️ Scope Reminder

* No Collections (`ArrayList`, `Map`, etc.)
* No advanced OOP beyond current level
* Console-only application
* Focus on arrays and matrices only
