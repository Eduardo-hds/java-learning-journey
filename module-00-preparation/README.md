# Java Bootcamp — Module 00: Java Platform Fundamentals & Setup

## Overview

This module is the foundation of the entire Java Bootcamp.

Before writing advanced Java code or working with frameworks, it is essential to understand how the Java platform works internally, how Java programs are executed, and how to properly set up a professional development environment.

The goal of this module is to build a strong conceptual base so that all future learning becomes clearer, faster, and more practical.

---

## Objectives

By the end of this module, I will understand:

- What Java is and how it evolved
- The history and philosophy behind Java
- The "Write Once, Run Anywhere" principle
- How the Java Virtual Machine (JVM) works
- The difference between JVM, JRE, and JDK
- The Java compilation and execution process
- What bytecode is and why it exists
- How Java memory management works (overview)
- Project structure in Java applications
- Package organization and naming conventions
- How to set up a professional Java development environment
- How to use IntelliJ IDEA effectively
- How to create and run Java projects
- How to debug Java applications
- How to read and use official Java documentation
- How to write and execute the first Java program

---

## Topics Covered

### 1. Java Platform Overview

- What Java is (language vs platform)
- History of Java
- Oracle and OpenJDK ecosystem
- Real-world usage of Java

### 2. Internal Architecture

- JVM (Java Virtual Machine)
- Bytecode concept
- Class loading process
- Basic memory model (Stack, Heap, Metaspace)
- Execution lifecycle of a Java program

### 3. JDK, JRE, and JVM

- Differences and responsibilities
- Tools included in the JDK
- How they work together

### 4. Development Environment Setup

- Installing Java (JDK)
- Environment variables (JAVA_HOME, PATH)
- Verifying installation

### 5. IntelliJ IDEA Essentials

- Project creation
- Running Java applications
- IDE navigation
- Debugging tools

### 6. Java Project Structure

- Standard directory layout
- Packages and naming conventions
- Clean code organization

### 7. Coding Standards

- Naming conventions (classes, methods, variables)
- Code readability principles

### 8. First Java Program

- Writing a basic `main` method
- Understanding program execution flow
- Compiling and running Java code

### 9. Debugging Basics

- Breakpoints
- Step-by-step execution
- Inspecting variables and call stack

### 10. Official Documentation

- How to read Java API documentation
- How to search for classes and methods

## Learning Methodology

This module follows a structured, professional learning approach:

- Every concept is explained with:
  - What it is
  - Why it exists
  - How it works internally
  - When to use it
  - Best practices
  - Common mistakes

- Practical exercises will be included in every chapter
- No topic will be skipped without full understanding
- The focus is on clarity and real-world application
