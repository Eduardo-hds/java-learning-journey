# 🏁 Chapter 8 — Final Projects (3 Progressive Systems)

Instead of a single project, you will build **three complete OOP projects**, each one reinforcing different aspects of Object-Oriented Programming.

Each project will be built progressively, step by step, not all at once.

---

## 💼 Project 1 — Student Management System

### 🎯 Goal

Practice basic OOP structure and object interaction.

### 📦 Core Features

* Create students
* Store name, age, and grade
* Display student information
* Simple in-memory management (no database)

### 🧱 Focus Skills

* Classes and objects
* Constructors
* Methods
* Encapsulation basics

---

## 🏦 Project 2 — Bank System

### 🎯 Goal

Strengthen encapsulation and business logic.

### 📦 Core Features

* Bank accounts
* Deposit method
* Withdraw method
* Balance validation (no negative balance)
* Account display info

### 🧱 Focus Skills

* Encapsulation (private fields)
* Getters and setters
* Data validation
* Method logic and state changes

---

## 📦 Project 3 — Inventory System

### 🎯 Goal

Combine multiple classes and object interaction.

### 📦 Core Features

* Products with name, price, quantity
* Add/remove stock
* Calculate total inventory value
* Basic inventory management logic

### 🧱 Focus Skills

* Multiple objects working together
* Clean class separation
* Constructors + encapsulation
* Real-world modeling

---

# 🏗️ Project Rules (IMPORTANT)

For all 3 projects:

* Build **one at a time**
* Do NOT jump ahead
* Always start simple, then improve
* Use proper package structure:

  * `model`
  * `service`
  * `util` (if needed)
* `Main.java` must stay clean and minimal
* No frameworks, no advanced features
