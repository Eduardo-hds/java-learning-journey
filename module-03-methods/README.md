# Bootcamp Java — Module 03: Methods

Welcome to **Module 03**.

From this point forward, you'll stop writing everything inside `main()` and begin organizing programs into **small, reusable methods**.

This is one of the biggest milestones in programming because methods allow you to:

* eliminate duplicated code;
* make programs easier to read;
* make code easier to test;
* divide complex problems into smaller ones;
* prepare for Object-Oriented Programming later.

As requested, we'll **not** introduce OOP concepts yet. Every example will remain procedural using `static` methods and console applications.

# Learning Goals

By the end of this module, you will be able to:

* Design reusable methods.
* Correctly declare and invoke methods.
* Pass parameters and return values.
* Understand variable scope.
* Choose between `void` and returning methods.
* Use method overloading appropriately.
* Work with variable-length arguments (`varargs`).
* Understand and implement recursion.
* Organize larger console applications into small, maintainable pieces.
