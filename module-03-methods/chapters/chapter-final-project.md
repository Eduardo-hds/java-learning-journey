# Final Project — Utility Toolkit

You have now learned everything required to complete the final project.

Build a **console-based Utility Toolkit** using **reusable methods**.

The application should include:

### Calculator

* Addition
* Subtraction
* Multiplication
* Division

### Temperature Converter

* Celsius ↔ Fahrenheit
* Celsius ↔ Kelvin

### Number Analyzer

* Even or Odd
* Positive / Negative / Zero
* Prime Number

### Factorial

* Iterative version
* Recursive version

### Fibonacci

* Generate the sequence for a chosen number of terms

### Menu System

Your program should use methods such as:

```text
main()

↓

showMenu()

↓

readOption()

↓

processOption()

↓

calculatorMenu()

temperatureMenu()

numberAnalyzerMenu()

factorialMenu()

fibonacciMenu()
```

The goal is to make `main()` as small and readable as possible.

Do **not** put all the logic inside a single method.
