# 🔁 Key Comparison Preview (Abstract Class vs Interface)

| Feature        | Abstract Class       | Interface                               |
| -------------- | -------------------- | --------------------------------------- |
| State (fields) | Yes                  | No (only constants)                     |
| Constructors   | Yes                  | No                                      |
| Method types   | Abstract + concrete  | Mostly abstract (default methods exist) |
| Inheritance    | Single               | Multiple                                |
| Purpose        | Shared base behavior | Contract / capability                   |
